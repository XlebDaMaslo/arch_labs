lab4:
MAX_SIZE: Это макрос, определяющий максимальный размер массива. В данном случае он равен 100.
divided_diff: Это функция, которая вычисляет разделенную разность для данного набора значений x и y. Принимает на вход массивы x и y, а также их размер n.
x: Указатель на массив значений x.
y: Указатель на массив значений y.
n: Количество точек для интерполяции.
result: Переменная для хранения результата интерполяции.
term: Переменная, используемая в методе Ньютона для вычисления нового значения.
file: Указатель на файл, который будет открыт и из которого будут считываться данные.
x_val: Переменная для временного хранения считываемого значения x.
i: Переменная для использования в циклах в качестве счетчика.
x_value: Значение x, для которого будет вычислен результат интерполяции.
result: Результат интерполяции, который будет выведен на экран.

lab6:
a: Начало отрезка интегрирования.
b: Конец отрезка интегрирования.
n: Количество разбиений отрезка [a, b] для метода трапеций.
h: Шаг интегрирования, вычисляется как (b - a) / n.
sum: Переменная для накопления суммы значений функции в точках разбиения.
x: Текущая координата x внутри цикла.
epsilon: Заданная пользователем погрешность для метода с контролем погрешности.
I1, I2: Переменные для хранения результатов интегрирования с разным количеством разбиений (используются для сравнения и проверки условия погрешности).
choice: Выбор пользователя (1 - задать N, 2 - задать epsilon).
